🏢 SharePoint Setup Required SharePoint's security policy blocks direct calls to external APIs. To run this tool inside SharePoint, you need a small proxy. Follow these steps:
  1. Deploy the one-file proxy to Azure (free tier) using proxy-server.js and SHAREPOINT-DEPLOYMENT.md included in this download package — or ask your IT team to host it. Full Azure CLI commands are in the deployment guide.
  2. Once deployed, paste your proxy URL below (e.g. https://myproxy.azurewebsites.net/v1/messages).
  3. Enter your Anthropic API key — it stays in this browser tab only, never sent to SharePoint servers.
  4. Embed this HTML file in a SharePoint page using Edit → Embed web part, or upload to a Document Library and link it.

⚠️ Heads up — this looks like a PubMed abstract link.
PubMed abstract pages (pubmed.ncbi.nlm.nih.gov) only show a short summary paragraph, not the full paper. The tool needs the full text to work properly.

Two better options:
• Use the PubMed Central link instead — it starts with pmc.ncbi.nlm.nih.gov and contains the full open-access article.
• Or switch to the Paste Text tab and copy-paste the abstract or full text directly.

✅ Best sources: PubMed Central (pmc.ncbi.nlm.nih.gov), bioRxiv, medRxiv, or any open-access full-text page.
❌ Avoid: Standard PubMed abstract links — these don't include full paper text.
